## Agen (Instructions for Agents)

- **Garis Pembatas (Border Boundaries)**: Seluruh konten website (termasuk navbar, footer, dsb) harus berada di dalam garis pembatas (border-x) dengan lebar `max-w-7xl` yang telah ditentukan di layout utama.
- **Interaktivitas & Konten**: Jika membuat interaktivitas atau elemen baru, pastikan tetap berada di dalam area garis pembatas tersebut. 
- **Efek Visual**: Elemen dekoratif atau efek (seperti blur atau gradasi) boleh melampaui garis pembatas asalkan elemen utamanya tetap di dalam. Jika ada elemen yang harus berada di luar, pastikan elemen tersebut tersembunyi (`opacity-0` atau `hidden`) jika tidak diperlukan atau dibuat tidak terlihat.
- **GSAP & Tailwind**: Lihat file `.gsap` dan `.tailwind` di root repository untuk panduan cara menulis kode animasi dan styling yang konsisten dengan desain sistem ini.
- **SEO & Performa**: Gunakan tag HTML semantik (h1, section, main, dsb) untuk SEO yang baik dan pastikan tidak ada redudansi kode untuk performa maksimal.
